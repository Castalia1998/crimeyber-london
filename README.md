# C(rime)YBER LONDON

A computational urban project developed for Digital Futures 2025:  
**C(rime)YBER LONDON: A Graph Neural Network Approach to Urban Crime Prediction**

## Overview
This project explores urban crime prediction through a graph neural network framework, developed in the context of Digital Futures 2025. It combines computational urban analysis, graph-based thinking, and speculative visual presentation.

## My Role
- Co-developed the project concept and overall framework
- Led project coordination and communication with instructors
- Collected, selected, and processed raw urban data into model-ready inputs
- Directed final visualization and presentation strategy
- Produced presentation materials including video, website, and slide deck

## Current Status
Completed as a Digital Futures 2025 workshop project.
